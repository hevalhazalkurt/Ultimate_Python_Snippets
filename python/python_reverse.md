### `reverse()`

#### What is `reverse()` for?

The `reverse()` method reverses the elements of a given list.

**Syntax**

```python
list.reverse()
```


<br> <br>


**Example 1**

```python
names = ["Sally", "Barry", "Chuck", "Steven"]
print(names)

names.reverse()
print(names)
```

**Output**

```
['Sally', 'Barry', 'Chuck', 'Steven']
['Steven', 'Chuck', 'Barry', 'Sally']
```


<br><br>

**Example 2**

```python
numbers = [1, 2, 3, 4, 1, 2, 6]
print(numbers)

numbers.reverse()
print(numbers)
```

**Output**

```
[1, 2, 3, 4, 1, 2, 6]
[6, 2, 1, 4, 3, 2, 1]
```
